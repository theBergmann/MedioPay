Install the plugin:

copy the folder here:
wp-content/plugins

(directory should be wp-content/plugins/Bitcoin-SV-Paywall-Plugin-for-Wordpress)

After installing and activating plugin you will able to use "[cp-hide-button]" (without quote)

For your example:

[cp-hide-button to="######" amount="1.00" currency="BSV"]
 This is where you can add custom html content, which will get hidden.
[/cp-hide-button]
